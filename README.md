# libindy-common-java

## Information

This is a shared library for use by Hyperledger Indy based components.

## Maven

Just run:

	mvn clean install

Dependency:

	<dependency>
		<groupId>com.danubetech</groupId>
		<artifactId>libindy-common-java</artifactId>
		<version>0.2.0</version>
		<scope>compile</scope>
	</dependency>
